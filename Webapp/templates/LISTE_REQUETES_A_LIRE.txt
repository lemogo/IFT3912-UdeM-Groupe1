Listes variables "globales" à générer

J'utilise le terme variables globales, juste pour dire qu'elles doient être générées dans tous les render de templates.
Les variables sont faciles à voir. Simplement ouvrir un fichier template en particulier pour une requête et regarder.
Cependant, il y a quelques cas spéciaux :

user				//Retourne vrai si le user est connecté, faux sinon.
notifications_number 		//Nombre de notifications. Doit être accessible en tout temps lorsque le user est connecté.
